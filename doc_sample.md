## 操作フロー

```mermaid
flowchart TD
  A([トップページを開く]) --> B{ログイン済み？}
  B -- いいえ --> L[ログイン画面へリダイレクト]
  L --> A

  B -- はい --> C[トップページ表示<br/>左サイドバー / メイン領域]
  C --> D{左メニューを選択}

  %% ユーザー機能
  D -->|ユーザー機能| U1[ユーザー機能一覧を表示]
  U1 --> U2{操作を選択}
  U2 -->|検索| U3[条件入力 → 検索実行]
  U3 --> U4[検索結果一覧を表示]
  U4 --> U5{詳細を見る？}
  U5 -- はい --> U6[詳細画面へ遷移]
  U5 -- いいえ --> U1

  U2 -->|新規作成| U7[入力フォーム表示]
  U7 --> U8{入力チェックOK？}
  U8 -- いいえ --> U7
  U8 -- はい --> U9[登録処理]
  U9 --> U10[完了メッセージ表示]
  U10 --> U1

  %% 管理者設定
  D -->|管理者設定| M1[管理者設定メニュー表示]
  M1 --> M2{マスタを選択}
  M2 -->|福利厚生マスタ| W1[福利厚生マスタ一覧表示]
  W1 --> W2{操作を選択}
  W2 -->|追加| W3[福利厚生を追加（入力）]
  W3 --> W4{入力チェックOK？}
  W4 -- いいえ --> W3
  W4 -- はい --> W5[保存]
  W5 --> W6[反映・一覧更新]
  W6 --> W1

  W2 -->|編集| W7[対象行を選択 → 編集フォーム]
  W7 --> W5

  W2 -->|削除| W8{削除確認}
  W8 -- キャンセル --> W1
  W8 -- OK --> W9[削除実行 → 一覧更新]
  W9 --> W1

  %% 共通
  D -->|ログアウト| Z1[ログアウト]
  Z1 --> L
```
