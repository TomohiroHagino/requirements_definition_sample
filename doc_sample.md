## 操作フロー

```mermaid
flowchart TD
  A([トップページを開く]) --> B{ログイン済み？}
  B -- いいえ --> L[ログイン画面へリダイレクト]
  L --> A

  B -- はい --> C[トップページ表示<br/>左サイドバー / メイン領域]
  C --> D{左メニューを選択}

  %% ユーザー機能（例）
  D -->|ユーザー機能| U1[ユーザー機能一覧を表示]
  U1 --> U2{操作を選択}
  U2 -->|検索| U3[条件入力 → 検索実行]
  U3 --> U4[検索結果一覧を表示]
  U4 --> U5{詳細を見る？}
  U5 -- はい --> U6[詳細画面へ遷移]
  U5 -- いいえ --> U1

  U2 -->|新規作成| U7[入力フォーム表示]
  U7 --> U8{入力チェックOK？}
  U8 -- いいえ --> U7
  U8 -- はい --> U9[登録処理]
  U9 --> U10[完了メッセージ表示]
  U10 --> U1

  %% 管理者設定（マスタ名は出さない）
  D -->|管理者設定| M1[管理者設定メニュー表示]
  M1 --> M2{設定項目を選択}
  M2 --> M3[設定一覧表示]
  M3 --> M4{操作を選択}
  M4 -->|追加| M5[追加フォーム表示 → 入力]
  M5 --> M6{入力チェックOK？}
  M6 -- いいえ --> M5
  M6 -- はい --> M7[保存 → 一覧更新]
  M7 --> M3

  M4 -->|編集| M8[対象を選択 → 編集フォーム]
  M8 --> M7

  M4 -->|削除| M9{削除確認}
  M9 -- キャンセル --> M3
  M9 -- OK --> M10[削除実行 → 一覧更新]
  M10 --> M3

  %% 共通
  D -->|ログアウト| Z1[ログアウト]
  Z1 --> L
```
